### LogSentinel
# 📝 Giới thiệu
LogSentinel là một công cụ CLI giúp tự động hóa việc phân tích 
các file log từ các dịch vụ phổ biến (web server, SSH server) để tìm kiếm dấu hiệu của 
các cuộc tấn công như brute-force, quét lỗ hổng, v.v.
